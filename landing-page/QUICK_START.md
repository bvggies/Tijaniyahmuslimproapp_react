# Quick Start Guide

## Installation Steps

1. **Navigate to the landing-page directory:**
   ```bash
   cd landing-page
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm start
   ```

4. **Open your browser:**
   The app will automatically open at `http://localhost:3000`
